# Contributing

Thank you for your interest in contributing to Takeover Triage!

## Reporting Issues

Please include:

- Operating System
- Python Version
- Command used
- Verbose Output
- Expected behavior
- Actual behavior

## Feature Requests

Please open a GitHub Issue before implementing large features.

## Pull Requests

1. Fork the repository.
2. Create a feature branch.

```bash
git checkout -b feature/new-provider
```

3. Commit your changes.

```bash
git commit -m "Add provider detection"
```

4. Push your branch.

```bash
git push origin feature/new-provider
```

5. Open a Pull Request.

## Coding Style

- Follow PEP8.
- Keep functions modular.
- Document all new providers.
- Include comments for complex logic.

## Testing

Always test against authorized targets only.

Thank you for helping improve Takeover Triage.
