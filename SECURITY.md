# Security Policy

## Supported Versions

| Version | Supported |
|----------|-----------|
| Latest | ✅ |
| Older Releases | ❌ |

## Reporting Security Issues

If you discover a security issue in Takeover Triage itself (not a target scanned with it), please report it privately.

Please include:

- Version
- Operating System
- Python Version
- Proof of Concept
- Steps to Reproduce

Do not disclose vulnerabilities publicly until they have been reviewed.

## Responsible Use

This project is intended for:

- Bug Bounty Programs
- Penetration Testing
- Internal Security Assessments
- Research

Never use this tool against systems without authorization.
